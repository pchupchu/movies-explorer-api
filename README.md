# movies-explorer-api

Бэкенд для дипломной работы Яндекс Практикума

**Movies Explorer** - сервис, в котором можно найти фильмы по запросу и сохранить в личном кабинете.
<br>
## Доступные скрипты

### `npm start`

### `npm test`

### `npm run build`
